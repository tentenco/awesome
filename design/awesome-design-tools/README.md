## 出色的設計工具作為 Flawless App 系列的一部分加入 Abstract！

今天，我們很高興地宣布，Flawless App 已加入[Abstract](https://www.abstract.com)，這是一個設計交付平台，可為設計帶來可見性、責任性、可測量性和可預測性。

Flawless App 是我們的第一家公司，我們為我們的 5 人團隊所取得的一切感到自豪。

自2015 年以來，我們為設計師和開發人員推出了強大的工具- 其中包括[Flawless App](https://flawlessapp.io/)、[Reduce](https://flawlessapp.io/reduceapp)、 [Flawless Feedback] ]（https://flawlessapp.io/feedback）。我們也為社區驅動的舉措投入了大量的愛心和關懷。很棒的設計工具就是其中之一。

＃＃＃＃ 下一個是什麼？
我們的團隊已加入 Abstract，專注於建置 [Abstract SDK](https://sdk.goabstract.com)，以提升開發人員的體驗。一段令人難以置信的旅程已經結束，新的旅程才剛開始。如果您想進一步了解 Awesome Design Tools 的未來，[請繼續閱讀我們的常見問題解答](https://flawlessapp.io/designtools/joinsabstract)。

### 設計交接工具

- [Fathom](https://usefathom.com/) — 提供簡單、有用的網站統計信息，無需追蹤或儲存使用者的個人資料。
- [FullStory](https://www.fullstory.com/) — 一款可在一個強大且易於使用的平台中擷取所有客戶體驗資料的應用程式。

- [Heap](https://heapanalytics.com/) — 自動擷取每個 Web、行動和雲端交互，然後追溯分析您的數據，而無需編寫程式碼。
- [Heatly](https://www.heatlyanalytics.com) — 免費熱圖工具，可以更了解網站的使用者體驗。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg)

- [LiveSession](https://livesession.io) - 高品質的使用者會話重播分析。

### 漸層工具

### 原型設計工具

原型是所提出的解決方案的簡單實驗設計。它應該有助於以快速且廉價的方式測試想法、設計假設和假設。原型設計工具使設計人員和客戶能夠了解產品在現實世界中的功能，並就該解決方案進行協作。許多現代原型設計工具可用於線框圖、原型設計和協作：

- [Alva](https://www.meetalva.io/) — 使用程式碼元件建立活動原型。它也是開源的。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg) [![open-source.svg](https://github.com /LisaDziuba/Awesome-Design-Tools/blob/master/Media/open-source.svg)](https://github.com/meetalva/alva)

- [Survey Monkey](https://www.surveymonkey.com/) — 線上調查工具，用於捕捉對您最重要的人的聲音和意見。
- [Typeform](https://www.typeform.com/) — 使用簡單的拖放介面建立任何類型的表單、調查或問卷，甚至接受信用卡付款。
- [Wufoo](https://www.wufoo.com) — 建立表單、收集資料和付款並自動化您的工作流程。
- [YesInsights](https://www.yesinsights.com/) — 簡單的問題和 NPS 調查可改善您的業務。
- [UserBit](https://userbitapp.com/) — 為您的團隊提供即時研究工具的平台。標記/編碼訪談和回饋、捕捉見解、創建角色、視覺化網站地圖等。
- [使用者訪談](https://www.userinterviews.com/) — 根據職業、人口統計、地理等因素，從擁有 125,000 名成員的社群中招募參與者，進行任何類型的研究。
- [Zoom](https://zoom.us) — 這是最好的線上會議服務之一。

### 線框工具

線框是一種視覺模型，概述了網站或應用程式的基本結構。它包含了最基本的元素和內容，可幫助您輕鬆解釋設計想法。線框是一種低保真度的設計展示方式。本節介紹了適用於各種用例的最佳線框圖工具。

- [Balsamiq Cloud](https://balsamiq.cloud/) — 輕鬆繪製 UI 草圖。捕捉您的想法，協作設計，讓每個人都參與其中。
- [BLOKK](http://www.blokkfont.com/) — 一種為不懂拉丁語的顧客提供快速模型和線框圖的字體。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg)
- [CSS Grid Builder](https://cssgrid.cc/css-grid-builder.html) — 使用此工具，網格中沒有 HTML 部分。只有在根據網格規則新增和定位內容（元素）時，網格才會變得明顯。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg)
- [Gliffy](https://www.gliffy.com/) — 用於建立框架、UML 圖、流程圖、線框圖等的工具。
- [Gridzzly](http://gridzzly.com/) — 建立可列印的自訂網格紙最簡單的方法。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg)
- [Layoutit](https://grid.layoutit.com/) — CSS Grid 和 Bootstrap 的介面建構器，希望成為前端開發的開始。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg)
- [Pidoco](https://pidoco.com/en) — 此軟體可讓您快速建立點擊式線框和完全互動式的 UX 原型。
- [Wireframe.cc](https://wireframe.cc/) — 一個簡單的線框工具，不會妨礙您。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg)
- [異想天開的線框](https://whimsical.co/wireframes/) — 即時線框觸手可及。豐富的可配置元素庫，如按鈕、輸入、複選框。
- [1200px 網格系統](https://1200px.com/) — 幫助您建立比 960 像素樣式更廣泛的網站設計網格系統。






### 三維建模軟體

3D 圖形用於遊戲、電影製作、建築、工程和 3D 列印。3D 藝術家和設計師使用本節中提到的特定 3D 建模軟體。

- [Autodesk](https://www.autodesk.com/products/fusion-360/overview) — 整合 CAD、CAM 和 CAE 軟體。將設計、工程和製造統一到一個平台。

- [FreeCAD](https://www.freecadweb.org/) — 一款免費、開源的多平台 3D 參數化建模器。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg) ![open-source.svg](https://github.com/ LisaDziuba/Awesome-Design-Tools/blob/master/Media/open-source.svg)
- [MatterControl](https://www.matterhackers.com/store/l/mattercontrol/sk/MKZGTDW6) — 一款免費、開源、一體化軟體包，可讓您設計、切片、組織和管理您的3D列印作品。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg) ![open-source.svg](https://github.com/ LisaDziuba/Awesome-Design-Tools/blob/master/Media/open-source.svg)

- [Tinkercad](https://www.tinkercad.com) — 一款免費且易於使用的應用程序，適用於 3D 設計、電子產品和編碼。![free.svg](https://github.com/LisaDziuba/Awesome-Design-Tools/blob/master/Media/free.svg)
